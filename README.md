# Carbon-Footprint-PromptWars

An AI-powered carbon footprint calculator and lifestyle advisor.

## Features
- **Accurate Calculations**: Computes monthly and annual CO2 emissions based on lifestyle choices.
- **AI Advisor**: Provides customized action plans to reduce your footprint.
- **Simulator**: Simulates the exact reduction you would see by adopting specific green habits.

## Tech Stack
- Frontend: React + Vite
- Backend: FastAPI (Python)
- Deployment: Vercel
